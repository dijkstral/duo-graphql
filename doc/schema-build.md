# Schema构建

Schema由GraphQL-Engine根据注册进来的GraphQL-Providers信息自动完成。

需要的信息有：

- GraphQL-Provider服务信息。包括服务名、版本、领域名称、调用地址、子领域信息、暴露的服务信息。
- GraphQL-Provider接口信息。包括Spring MVC接口的各类信息。
